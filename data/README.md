# Dataset

This framework was developed and validated using the **Ireland Cyber Incident Dataset**.

## About the Dataset

The Ireland Cyber Incident Dataset contains 30 annotated cybersecurity incidents relevant to Irish SMEs and public sector organisations. It is the first structured incident dataset with an Irish-first focus, designed specifically for SOC optimisation, severity modelling, and incident response research.

Each incident record includes:
- Detection and response times (MTTD / MTTR)
- Severity classification (SEV1 / SEV2 / SEV3)
- Asset value, data sensitivity, user role, and alert timing scores
- Documentation and PIR completion status
- Analyst confidence level
- MITRE ATT&CK technique references
- Source reference (public advisory or composite)

## Note on Methodology

The dataset was compiled from publicly available Irish cybersecurity advisories, CERT-IE reports, academic case studies, and simulated attack scenarios representative of threats faced by Irish SMEs and public sector organisations. It is a structured simulation dataset intended for research and educational purposes, not a log of live production incidents.
